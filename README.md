# AmazonStyleFullStackMicroservices
Building Amazon Style Full Stack Microservices: Using .Net 9, Rabbit MQ, Saga Pattern, Outbox, Angular 20, Docker, Kubernetes, Azure, Service Mesh and many more...

+ create Microservices using .Net Core
+ implement message queues
+ implement Mediator Pattern
+ implement CQRS Pattern
+ Containerize entire solution
+ use Application Gateway
+ use Saga Pattern
+ use Outbox Pattern
+ implement structured Logging
+ handle cross cutting concerns like ELK Stack, Observability etc
+ put all the workloads on the K8s cluster
+ apply Service Mesh using Istio
+ implement Side Car pattern
